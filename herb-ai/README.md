# HerbID: AI-Powered Botanical Identifier & Medicinal Guide

An end-to-end Data Science project utilizing Computer Vision to classify medicinal herbs from images and retrieving their traditional/scientific usages using structured data.

## 🚀 Project Overview
- **Core Objective:** Classify herb species from image uploads and provide reliable medicinal context.
- **Tech Stack:** Python, PyTorch (or TensorFlow), FastAPI, SQL, React.

## 📂 Repository Structure
- `research/`: Jupyter notebooks exploring data augmentation, model training, and evaluation metrics (Confusion Matrix, F1-Score).
- `backend/`: FastAPI application serving model inference and querying the herb database.
- `frontend/`: React application providing an interface for image uploads.
